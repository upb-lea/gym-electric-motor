Polynomial Static Load¶

Mechanical ODE¶

\[\frac{ \mathrm{d} \omega_{me} } { \mathrm{d} t } = \frac{ T - T_L(\omega_{me})}{J_{total}}\]

Polynomial Load Equation¶

\[\begin{split}T_L(\omega_{me})=\mathrm{sign}(\omega_{me})(c \omega^2_{me} + b \vert\omega_{me}\vert + a)\\\end{split}\]

Class Description¶

class gym_electric_motor.physical_systems.mechanical_loads.PolynomialStaticLoad(load_parameter=(), limits=None, load_initializer=None, **__)[source]¶

Mechanical system that models the Mechanical-ODE based on a static polynomial load torque.

Parameter dictionary entries:
a: Constant Load Torque coefficient (for modeling static friction)
b: Linear Load Torque coefficient (for modeling sliding friction)
c: Quadratic Load Torque coefficient (for modeling air resistances)
j_load: Moment of inertia of the mechanical system.
